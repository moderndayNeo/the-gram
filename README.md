## Outline

The Gram is a fullstack clone of Instagram build with a React frontend and a Rails backend API. It uses a Postresql database combined with AWS S3 for hosting images.

## Technologies

Rails 6.0.3.4
Ruby 2.7.0
Node v12.14.0
Postgres (PostgreSQL) 12.5

React 17.0.1
Redux 4.0.5
AWS
